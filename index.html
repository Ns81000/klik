Built for precision

Every feature designed to make text automation effortless and reliable.

Global F6 Hotkey

Start and stop automation from anywhere on your system. No need to switch windows — press F6 and go.

Preset Manager

Save and recall named message and interval combinations. Build a library of automations you can switch between in seconds.

Custom Send Key

Capture any key as the send trigger. Enter ⏎ by default — remap to Tab, Space, or any key your workflow needs.

Precision Intervals

Set intervals in minutes, seconds, and milliseconds. A 10ms safety floor prevents input lockups while keeping timing sharp.

Live Status Panel

Monitor your automation in real time. Send counter, elapsed timer, and warning messages — everything visible at a glance.

System Theme Sync

Follows your Windows light or dark mode automatically. The UI stays native to your desktop without any manual toggling.
